# Project Goal Plan

## Objective
Build a production-grade, scalable DevOps pipeline for VM lifecycle management using Proxmox VE, triggered by GitHub Actions, and deploying a Python web app with a backend DB.

## Tools
- Terraform: Infrastructure provisioning (Hetzner, Proxmox)
- Ansible: Configuration, VM template, app deployment
- GitHub Actions: CI/CD pipeline automation
- Python (Flask/FastAPI): Web app and backend DB
- AI Coding Assistant: Script and documentation authoring

## Reach Goals
- Parameterized pipeline for multiple environments
- Automated monitoring and alerting
- Secure secrets management
- Horizontal scaling (multiple servers, parallel pipelines)
