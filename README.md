# DevOps MILES Challenge

This repository implements the DevOps challenge using Terraform, Ansible, GitHub Actions, and a Python web application.

## Structure
- `terraform/` — Infrastructure provisioning (Hetzner, Proxmox)
- `ansible/` — Configuration and deployment playbooks
- `app/` — Python web application (Flask/FastAPI)
- `.github/workflows/` — CI/CD pipeline
- `docs/` — Documentation, process logs, diagrams

## Steps
1. Provision infrastructure with Terraform
2. Configure Proxmox and create VM template with Ansible
3. Deploy Python app and DB to VM
4. Run smoke tests and collect logs
5. Tear down VM

## Deliverables
- Project goal plan
- Working solution (code, scripts, configs)
- Process documentation
- Reflection
